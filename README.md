# 🧠 What is CHRONATRACE?

CHRONATRACE is a next-generation forensic intelligence operating system designed to help investigators process fragmented evidence pipelines using AI-assisted reasoning, deterministic contradiction detection, timeline reconstruction, and investigation memory systems.

Instead of relying on a single monolithic AI model, CHRONATRACE combines:

- deterministic forensic logic
- retrieval-augmented reasoning (RAG)
- computer vision
- NLP extraction
- timeline intelligence
- modular multi-model AI routing

to create a scalable forensic investigation environment.

# 🚀 Getting Started

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/chronatrace
cd chronatrace
```

---

## 2️⃣ Install Frontend Dependencies

```bash
cd frontend
npm install
```

---

## 3️⃣ Run Frontend

```bash
npm run dev
```

Frontend:
```bash
http://localhost:3000
```

---

## 4️⃣ Setup Backend

```bash
cd backend
pip install -r requirements.txt
```

---

## 5️⃣ Run FastAPI Server

```bash
uvicorn main:app --reload
```

Backend:
```bash
http://localhost:8000
```
